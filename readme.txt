Modify WP Admin Branding and Color Scheme

== Description ==

Modify WP Admin Branding and Color Scheme. If provides options to change WP Admin Login Logo, WP Admin Login Header URL, WP Admin Login Header Title and WP Admin Login Page Color Scheme

== Frequently Asked Questions ==

= Does it have option to change WP Admin Login Logo =

Yes you can add the logo to WP media and add its URL to the plugin options. 

= What do we have to enter in Color Field =

It is a Color picker field.
